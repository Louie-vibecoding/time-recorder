# 跨端手动测试矩阵 (Manual Cross-Platform Test Matrix)

发布每个版本前，在 **PC 和 Android 两端** 都跑一遍这些场景。全部打勾 = 可以发布。

> 注：进行中（未结束）的段，结束时间记作 `ing`；旧记录里用 `00:00` 表示进行中的也能正确读出。

## 准备 Setup
- [ ] PC：插件目录已同步（Obsidian Sync / 坚果云 / Syncthing 等）
- [ ] Android：插件目录已同步、`.obsidian/plugins/time-recorder/` 里能看到文件
- [ ] 两端：第三方插件里已启用 Time Recorder
- [ ] 两端：今日记录文件已存在或被自动创建（启动时自动建）

## A. 打卡流程 Punch-in flow

### A1. 当天第一次打卡（文件为空 / 新建）
- [ ] PC：开九宫格 → 点「学习」→ 弹 Notice → 检查文件出现 `- [ ] HH:MM - ing 学习`
- [ ] Android：同上

### A2. 后续打卡（封口上一段 + 追加新段）
- [ ] PC：开九宫格 → 点「通勤」→ 检查：学习段结束时间被填上、通勤段为 `ing`
- [ ] Android：同上

### A3. 自定义活动（只输入活动名，分类按名自动推断，无需手选）
- [ ] PC：✏️ 自定义 → 输入「做插件设计」→ 点「确定打卡」→ 文件追加该段（分类自动归类）
- [ ] Android：同上

### A4. 撤销
- [ ] PC：↶ 撤销 → 恢复到上一步状态
- [ ] Android：同上

## B. 时间轴视图 Timeline view

### B1. 渲染
- [ ] PC：开时间轴 → 24h 网格 + 已有色块正常渲染
- [ ] Android：同上；滚动顺滑

### B2. 点空白处
- [ ] PC：点 13:00 附近空白 → 编辑器打开、带智能填空的起止范围
- [ ] Android：同上

### B3. 编辑色块
- [ ] PC：点已有色块 → 编辑器打开并预填 → 改时间 → 保存 → 色块更新
- [ ] Android：同上

### B4. 删除色块
- [ ] PC：点色块 → 删除 → 色块消失
- [ ] Android：同上

### B5. 拖拽手柄（仅桌面）
- [ ] PC：拖色块上边缘 → 改时长、吸附到 5 分钟、松手落盘
- [ ] PC：拖色块下边缘 → 同上
- [ ] PC：进行中（`ing`）段不显示手柄、仍可点击编辑
- [ ] Android：手柄不显示/不可用 —— 点击编辑作为兜底仍可用

## C. 今日汇总 Today summary

### C1. 渲染
- [ ] PC：右侧栏打开汇总视图 → 按当前分类出表
- [ ] Android：命令面板 / ribbon 打开 → 全屏渲染

### C2. 自动刷新
- [ ] PC：打卡 → 汇总在 2 秒内更新（无需手动刷新；有轻微延迟属正常）
- [ ] Android：同上

### C3. 复制汇总
- [ ] PC：📋 复制汇总文本 → Notice → 粘到新笔记 → 出现 Markdown 表
- [ ] Android：同上

## D. 状态指示 Status indicator

### D1. PC 状态栏
- [ ] 底部状态栏显示当前活动
- [ ] 每 30 秒更新一次

### D2. Android ribbon 提示
- [ ] 长按 ribbon 图标 → 提示显示当前活动（注：小逸设备上 ribbon 是右划侧栏，方向差异不影响功能）

## E. 跨设备同步 Cross-device sync

### E1. PC → Android
- [ ] PC 打卡 → 等 10 秒 → Android 打开 Obsidian → 今日汇总显示新段

### E2. Android → PC
- [ ] Android 打卡 → 等 10 秒 → PC 今日汇总显示新段

## F. 设置 + 启动 Settings + bootstrap

### F1. 插件以默认设置加载
- [ ] 全新安装 → 设置页显示 10 个默认分类 + 其他

### F2. 可编辑设置（in-app）
- [ ] 设置页改记录文件夹路径 → 打卡 → 文件落到新文件夹
- [ ] 设置页新增一个分类 → 九宫格出现该分类 → 用它打卡正常
- [ ] 设置页给某分类加一个关键词 → 用含该词的自定义活动打卡 → 汇总归到该分类
- [ ] 设置页删除一个分类 → 九宫格不再显示它；历史记录里该活动归到「其他」（分类按名推断、不动历史）
- [ ] 设置页「恢复默认分类」→ 确认弹窗 → 恢复成 10 默认分类

### F3. 自动建今日文件
- [ ] 删掉今日文件 → 重载插件 → 文件被重新创建

### F4. 数据守卫（版本守卫）
- [ ] 手动把 `data.json` 改成非法 JSON（如删个括号）→ 重载插件 → 弹友好提示、回退默认设置、生成 `data.json.bak` 备份（插件不崩）

## 通过标准 Pass criteria

所有勾全部打上 = 可以发布。
