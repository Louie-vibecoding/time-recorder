# 时间记录仪 (Time Recorder)

让你 **两次点击就能完成一次打卡** 的 Obsidian 时间记录插件。可实时打卡、可睡前补录、自动出今日汇总表。灵感来自爱时间、时间日志等 App，但所有数据都在你自己的 Markdown 文件里。

支持 **桌面端和 Android 端**，同一套代码。

[English →](./README.md)

## 功能

- **九宫格打卡**：打开 Modal，点一个分类 —— 上一段自动封口、新一段开始。两次点击搞定。
- **自定义活动**：自由文本输入活动名 + 选一个分类用于汇总。
- **时间轴回填**：垂直 24 小时时间轴。点空白处新建段（智能填空到下一段），点已有块编辑或删除。桌面端可拖拽块的上/下边缘改时长（吸附到 5 分钟）。
- **今日汇总**：桌面侧边栏 / Android 全屏视图，按分类显示汇总。一键复制成 Markdown 表，方便贴到反省日志或交给 LLM 做深度分析。
- **1 步撤销**：最近一次打卡可一键还原。
- **状态栏 / Ribbon 提示**：随时显示「现在在做什么」（桌面在底部状态栏，Android 长按 ribbon 图标看提示）。
- **可编辑设置**：分类的增 / 删 / 改名、每类关键词、记录文件夹路径，全部在设置页里改，不用手动编辑文件。随时可一键恢复默认。
- **稳健的数据守卫**：万一设置文件损坏，插件会自动回退到默认设置、把损坏文件备份成 `data.json.bak`、并用大白话提示你 —— 绝不悄悄丢数据，也不会把另一台设备上更新版写的设置降级覆盖掉。
- **启动自动建今日文件**：打开 Obsidian 就能直接打卡。

## 安装

### 通过社区插件（上架后）

1. Obsidian → 设置 → 第三方插件 → 浏览
2. 搜索 "Time Recorder"
3. 安装 → 启用

### 手动安装

1. 从 [Release 页](https://github.com/<your-username>/time-recorder/releases) 下载 `manifest.json`、`main.js`、`styles.css`
2. 放到 `<你的 vault>/.obsidian/plugins/time-recorder/`
3. 重载 Obsidian → 启用

### Android 推荐设置

1. 设置 → 移动端 → 工具栏（或直接用左侧 ribbon 图标）
2. 把 "Time Recorder: Punch in" 加到第一个位置

> **Android 同步坑（重要）**：Obsidian Sync 默认 **不会** 同步第三方插件。要在电脑端打开 设置 → Sync → 同步配置 → 已安装的第三方插件，并在 Android 端打开 **同一个** 开关，云端插件文件才会下到手机。另外记得在 Sync 排除项里加上 `.obsidian/plugins/time-recorder/node_modules`，避免同步上百 MB 的开发目录。

## 文件格式

每天一份记录文件 `<记录文件夹>/{YYYY-M-D} 时间记录.md`。每一段是一行：

```
- [ ] 08:30 - 10:00 学习obsidian
```

- 结束时间写 `ing` 表示「还在进行中」。旧记录里用 `00:00` 表示进行中的，也能正确读出来。
- 已有的同格式手写记录无需修改即可解析。
- 分类是**读取时按活动名现推断**的（不写进文件），所以分类的增删改名都不会动到你的历史记录。

## 设置项

| 设置 | 默认 | 说明 |
|---|---|---|
| 记录文件夹 | `反省日志/时间记录` | 每日文件存放位置 |
| 模板路径 | `反省日志/时间记录/timer template.md` | 新建文件时复制的模板（可选） |
| 分类 | 10 个默认分类 + 其他 | 在设置页直接增删改名、编辑每类关键词 |

默认分类：😴 睡眠 / 📚 学习 / 🍱 饮食 / 🚿 个人卫生 / 🚗 通勤 / 💬 社交 / 🧹 家务 / 💼 工作 / 🏃 运动 / 🎮 娱乐，外加一个固定的 ❓ 其他 兜底分类。
